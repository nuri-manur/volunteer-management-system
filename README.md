# Volunteer Management System (Java)

Console application designed to manage volunteer databases efficiently. Created as my first major project in Java development.

## Features
- **Full CRUD Support**: Add, view, edit, and delete volunteer records.
- **Persistence**: Automatically saves and loads data from a local `.txt` file.
- **Advanced Search**: Find volunteers by their **Skills** or **City**.
- **Data Analytics**: Calculates total count and average age of participants.
- **Robustness**: Implemented `try-catch` blocks to handle invalid user inputs.
- **Clean UI**: Formatted table output for better readability.

## Tech Stack
- **Language**: Java
- **Concepts**: Collections (ArrayList), File I/O, Exceptions, Lambda expressions.
